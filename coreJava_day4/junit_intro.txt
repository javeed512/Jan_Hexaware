Testing:
=============
Testing helps developers to ensure quality of software/application.

1. Find bugs
2. Check Software Quality
3. Ensure Software implemented as per client requirement
4. Verify Robust 


Unit : method , function , class , module , webpage

Unit Testing is concept where a single  unit will be tested by applying set of inputs or sequence of inputs(test cases) by developers.
